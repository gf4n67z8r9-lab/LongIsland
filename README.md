# Long Island Yacht Charters

A modern, responsive website for Long Island Yacht Charters - your premier destination for water sports adventures!

## 🌊 Features

- **Responsive Design** - Works perfectly on desktop, tablet, and mobile
- **Modern UI** - Clean, professional design with smooth animations
- **Fast Loading** - Optimized performance with no dependencies
- **Easy to Customize** - Simple HTML, CSS, and JavaScript
- **Free Hosting** - Live on GitHub Pages

## 📋 Website Sections

1. **Navigation** - Sticky header with smooth scrolling
2. **Hero** - Eye-catching banner with call-to-action
3. **About** - Information about the yacht and services
4. **Services** - Three main offerings with pricing:
   - 🪁 Kite Surfing ($150/person)
   - 🤿 Scuba Diving ($120/person)

## ✏️ Customization

### Update Business Information

Edit `index.html` and replace:
- Email: `info@longislandyachtcharters.com`
- Phone: `+1 (555) 123-4567`
- Location: `Long Island Marina`
- Hours: `Daily 8:00 AM - 6:00 PM`

### Update Pricing

Find the Services section and modify:
```html
<p class="price">From $150/person</p>
```

### Add Your Images

Replace image URLs in the Gallery section with your own or from:
- [Unsplash](https://unsplash.com) - Free high-quality images
- [Pexels](https://www.pexels.com) - Free stock photos

### Change Colors

Edit the `:root` CSS variables in `index.html`:
```css
:root {
    --primary-color: #0066cc;      /* Main blue */
    --secondary-color: #00b4ff;    /* Light blue */
    --accent-color: #ff6b35;       /* Orange */
}
```
